Part II. Introduction to Generative AI (Week 1.2)
1. What Is Generative AI?
Generative AI: systems that generate complex, structured outputs, such as: - Text
(words) - Images (pixels) - Speech (audio samples) - Video (spatiotemporal data)
Core challenge: - Output space is astronomically large - Brute-force generation is
impossible



2. Generative AI vs. Classification
Classification (not GenAI problem)          Generative AI
Choose from fixed labels                    Create new content
e.g., Spam / not spam classification        e.g., Write an email
e.g., Cat / dog detection in pictures       e.g., Generate a new image

Key distinction: Generative AI produces outputs that did not exist in the training data.



3. AI → ML → Deep Learning → Generative AI
Understand the hierarchy:

   •   AI: the goal (ask machine to have intelligent behavior)
   •   Machine Learning: method (learn from data)
   •   Deep Learning: method (powerful ML using neural networks)
   •   Generative AI: a goal often achieved via deep learning
Most modern GenAI systems: - Are deep neural networks - Have millions to billions of
parameters



4. Models as Functions
A key mental model used in this course:
       A GenAI model is a function with a very large number of parameters.
Examples:
- ChatGPT: function mapping text → text
- Image generators: function mapping text → image
Training = learning parameters
Inference = applying the function



5. Language Models & Word Prediction
Language models work by:
- Breaking text into smaller units (tokens)
- Predicting the next token based on context
- Repeating this process sequentially
This is called autoregressive generation.
Important insight: Even though generation feels creative, it is fundamentally structured
prediction.



6. Why GenAI Feels “Creative”
   •     Models generate combinations never seen verbatim during training
   •     They operate in extremely high-dimensional spaces
   •     Creativity emerges from probabilistic prediction, not intention


7. Why Today’s GenAI Is Different
GenAI is not new — what changed: - Scale (data + parameters) - Transformers - General-
purpose models
Shift from: - Specialists (fixed-function systems) - To generalists (one model, many tasks)
You tell the model what you want instead of switching tools.
8. Prompt Engineering
If you cannot change the model: - Change how you talk to it
Prompt engineering is: - The art of communicating with AI systems - Adding clarity,
structure, and context
     Instead of adjusting the model, adjust the human.


9. Two Ways to Improve AI Systems
   1. Train yourself
         o Better prompts
         o Better task decomposition
   2. Train or adapt models
         o Fine-tuning the open-source models (e.g., LLaMA)
This course covers both paths.
